O que é um caractere invisível (ㅤ)?

Um caractere invisível (ㅤ) é um espaço em branco ou caractere Unicode não visível ao olho humano. No entanto, computadores e outros dispositivos podem detectá-los. Eles também são conhecidos como espaço invisível, letra invisível ou espaço vazio.

Como a maioria dos aplicativos não reconhece o espaço como um caractere, usamos esse caractere invisível, que adiciona um espaço em branco entre o texto ou dá a ilusão de que esse espaço invisível também é um caractere.

Coleção de vários Espaço em Branco

O Unicode tem mais de 10.000 espaços invisíveis, incluindo U+0020, U+2800, U+3164 e mais. Todos eles têm significados específicos e comprimentos diferentes para várias aplicações.

Vários usos de Letra Invisível

A Letra Invisível tem muitos usos na era digital de hoje. Listamos aqui alguns dos mais importantes:

1- Código HTML

Em códigos HTML, um espaço invisível é usado para criar uma aparência estruturada do programa. Por exemplo, o espaço sem quebra (&nbsp;) impede que uma quebra de linha ocorra entre palavras. Outras entidades não visíveis usadas em documentos HTML são &#8205; (Junção de Largura Zero), &#8204; (Não Junção de Largura Zero) e &#173; (Hífen Suave).

2- Senhas de Segurança

A ferramenta Espaço invisível ajuda a criar senhas de login seguras para suas contas. Quando você insere a senha, ninguém pode ver o que você está digitando porque os caracteres aparecerão em branco, mantendo suas contas seguras.

3- Mensagem invisível

Os aplicativos de mensagens de mídia social não permitem enviar uma letra invisível como uma mensagem de texto. Então, você pode copiar o espaço em branco da ferramenta de espaço invisível e usá-lo para enviar uma mensagem invisível engraçada para seus amigos e familiares.

4- Formatação da descrição do YouTube

Descrições do YouTube, especialmente para fins educacionais, devem ser escritas de forma organizada. Você pode formatar diferentes partes da descrição usando espaço em branco em seções separadas. 

5- Formatação de Documento

Criar um documento polido e profissional requer espaçamento e alinhamento precisos. Adicionar texto invisível no documento pode adquirir a vista perfeita e maximizar a legibilidade.

6- Nome de usuário em branco (Jogos e Mídias Sociais)

Você pode adicionar espaço em branco para criar um nome invisível exclusivo e excepcional para perfis de mídia social ou jogos (como Pubg, Free Fire, Call of Duty, Minecraft, etc). Espaço de Largura Zero (U+200B) é uma boa opção para criar um nick invisivel, pois muda a aparência, mas adiciona variação.

Perguntas Frequentes

Qual é o código alternativo para Preenchimento Hangul?

O código alternativo para Preenchimento Hangul é U+3164. Para inseri-lo no texto, segure a tecla Alt, digite 3164 no teclado numérico e solte a tecla Alt. Ou use uma ferramenta de texto invisível para copiar e colar o Preenchimento Hangul diretamente.

Qual é o significado de Unicode "U+2800"?

O Unicode "U+2800" representa o caractere "Padrão braille em branco". É usado na codificação braille para indicar uma célula vazia em uma célula braille.

Como Usar o Nbsp em HTML?

Nbsp é o caractere usado em HTML para adicionar um Espaço sem quebra. Você pode inseri-lo onde quiser, um espaço em branco que o navegador não removerá durante as quebras de linha.

Exemplo: Meu nome é Sammy Dower.

Qual é o uso deste símbolo ￼?

￼ é um Caractere de substituição de objeto. Quando o documento é convertido para texto simples, este símbolo representa objetos incorporados em um documento.
