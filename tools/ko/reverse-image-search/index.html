역 사진 검색이란 무엇이며방법 이미지 검색엔진 작동?

간단히, 비슷한 이미지 검색은 쿼리를 기반으로 정보를 찾기 위해 텍스트 기반 또는 음성 기반 키워드를 입력하는 대신 이미지를 업로드하는 일종의 온라인 검색입니다.

Google 이미지를 사용하면 웹에서 시각적으로 유사한 이미지를 신속하게 발견하고 객체 나 장소를 포함한 사진 및 객체 이름과 같은 메타 데이터에 대한 관련 정보를 얻을 수 있습니다.

예를 들어, 위의 귀여운 강아지 이미지를 업로드하면 엔진은 다음과 같이 반환됩니다.

기술적으로 말하면, 반전 (그림 검색)은 CBIR (콘텐츠 기반 이미지 검색) (QBIC) 및 CBVIR (콘텐츠 기반 시각 정보 검색)이라고하는 쿼리 기술을 사용하여 컴퓨터 비전을 적용함으로써 작동합니다. 수학적 모델을 기반으로 인터넷에서 디지털 이미지를 검색합니다. 구글의 경우도 마찬가지입니다 이미지 로 검색 하기. 정보를 검색하기 위해 샘플 이미지는 검색어를 공식화하므로 사용자가 키워드를 추측 할 필요가 없습니다. 반전 이미지 검색에 대한 나머지를 읽으십시오.

사진 조회를 역전시키는 많은 기술이 있지만, 그것들을 따로 다룰 필요는 없습니다. 오히려 사진 검색으로 얻을 수있는 것을 살펴 보자.
