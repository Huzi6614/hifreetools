RGB e HEX explicado!

O que é RGB, de qualquer maneira? RGB significa vermelho, verde e azul. Estas são as cores primárias da luz e formam o modelo de cores RGB.

O modelo de cor RGB é um sistema de cores de aditivo (em oposição a subtractiva) em que as luzes vermelhas, verdes e azuis são adicionadas em conjunto em diferentes proporções para reproduzir uma vasta gama de outras cores.

valores RGB geralmente vêm em uma escala de 0-255 e muitas vezes parecido com este: rgb (0, 74, 255).

Por outro lado, um código de cor hexadecimal é uma maneira de especificar cores usando valores hexadecimais.

O código em si é um trio hex, o que significa que ele representa três valores separados que indicam os níveis das cores componentes. Um valor de cor hexadecimal é composta de uma série de seis ou três caracteres precedidos por um hash (sinal de libra). A string muitas vezes consiste em 0-9 números e letras A-F.

Grandes exemplos são #FFFFFF que é o código hexadecimal para branco e # 000000 para o preto.

Enquanto modelo de cores RGB é usado principalmente para representar ou exibir gráficos em sistemas eletrônicos, como televisores, impressoras e computadores, hexadecimal é mais frequentemente usado para especificar as cores em HTML e CSS elementos movidos a código, como páginas da web.

Esta é praticamente a única diferença marcante entre os dois sistemas de cores. Fora isso, rgb e hex são apenas duas maneiras diferentes de expressar a mesma coisa: a valores de cores vermelhas, verdes e azuis.

MELHORES PRÁTICAS PARA TRABALHAR COM RGB e HEX sistemas de cores

  • valores de cor hexadecimais são na forma de uma única cadeia. Se você estiver trabalhando em uma equipe com engenheiros de software, tente usar números de cor hex como eles vão ser capazes de facilmente copiar e colar o código fora de um documento de chat ou e-mail.
  • Se você precisa para definir a opacidade de um objeto colorido, RGBA oferece essa funcionalidade. Você pode declarar valores de vermelho, verde e azul, e um valor adicional para a transparência.
  • códigos de cores hexadecimais são mais compactos e, portanto, são melhores para o código minimizado. Eles podem se tornar ainda mais compacto, utilizando apenas três dígitos para algumas cores (por exemplo #FFF), em vez de todos os seis dígitos (por exemplo, #FFFFFF).
  • Hex melhorar o tempo de carregamento de uma página web ligeiramente. Computadores lidar com dígitos binários, e um dígito hexadecimal podem ser facilmente convertidos em quatro dígitos binários, enquanto dígitos decimais levar mais tempo para converter.

Estes são, obviamente, algumas das razões para converter valores de cor RGB para Hex códigos.
