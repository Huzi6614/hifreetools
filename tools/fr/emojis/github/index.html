Github est l'une de ces plates-formes qui utilise la méthode d'insertion d'emoji, et elle est remplacée par le caractère d'encodage emoji natif. La plateforme ne dispose pas de shortcode standard pour l’encodage des emojis. On comprend que le code utilisé sur cette plateforme est différent de Slack dans de nombreux cas. Les codes emoji utilisés sont des caractères emoji non Unicode. Ils sont remplacés par le processus d'insertion de remplacement d'image de caractères. Github utilise les graphismes de celui qui était disponible dans le jeu vidéo doom de 1993. Des emojis personnalisés peuvent également être ajoutés à GitHub. Ici, vous pouvez trouver plusieurs emojis Github, et ils peuvent être utilisés en les copiant et en les collant là où vous souhaitez les ajouter sur le Plateforme GitHub.
