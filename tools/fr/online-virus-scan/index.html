Antivirus sur les serveurs Web

Les serveurs de réseau sont conçus et fonctionnent pour gérer le trafic vers les serveurs Web; les serveurs pare-feu contre les intrusions illégales protègent au réseau.

Ce sont les serveurs de pare-feu qui protègent les serveurs Web contre les logiciels malveillants, les virus et les attaques de pirates et autres tentatives d’intrusion illégale. les fournisseurs de services d’hébergement Web ne dépendent pas de l’installation d’un antivirus seulement sur leurs configurations.
