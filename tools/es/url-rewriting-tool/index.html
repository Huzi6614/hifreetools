ACERCA DE ESTÁTICA Y DINÁMICA WEBS

Usted puede encontrar algunos sitios web estáticos en internet. Un sitio web que está proporcionando la información sin ningún tipo de señales; es probable que un sitio web estático. Estos son comunes en las pequeñas empresas y empresas; lo que quieren es demostrar su presencia en Internet. Ellos no quieren que nadie robar su nombre de la empresa, y no hacen cualquier negocio en la web.

páginas web estáticas son generalmente más fáciles de acogida y de bajo costo para el desarrollo. Sin embargo, debe tener a alguien con conocimientos de HTML para actualizarlo.

Cuáles son las ventajas de los sitios web estática?

  • No es caro para desarrollar
  • Rápido y fácil de desarrollar
  • Rentable de acogida

Cuáles son las desventajas de los sitios web estática?

  • sólo puede proporcionar información básica
  • contenido web puede desencadenar
  • Requiere conocimientos web para modificar o actualizar

Cuáles son las ventajas de los sitios web dinámico?

  • página web dinámica tiene más funcionalidad
  • Muy fácil de operar y de actualización
  • Permite a los usuarios de Internet interactúan sin problemas
  • Permite el intercambio de información entre propietario del sitio web y los usuarios finales
